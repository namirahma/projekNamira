Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di browser untuk membuat halaman web menjadi interaktif.

Contoh Perintah JavaScript

  • console.log(): Menampilkan teks di konsol
  • document.getElementById(): Mengambil elemen berdasarkan ID
  • alert(): Menampilkan kotak pesan

Contoh Kode JavaScript

        console.log("Hello, World!");
        alert("Selamat Datang di JavaScript!");

        HASIL :
        Hello, World!
        Selamat Datang di JavaScript!
