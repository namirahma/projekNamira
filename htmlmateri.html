Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML menggunakan tag untuk menentukan elemen-elemen dalam halaman web.

Contoh Tag HTML

  • <html>: Tag utama untuk halaman HTML
  • <head>: Berisi informasi meta, link stylesheet, dll.
  • <body>: Berisi konten utama halaman
  • <p>: Membuat paragraf
  • <a>: Membuat hyperlink

Contoh Kode HTML

<!DOCTYPE html>
<html>
<head>
    <title>Contoh Halaman</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah paragraf pertama.</p>
</body>
</html>
