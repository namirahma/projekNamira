Pengenalan CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan elemen-elemen HTML, seperti warna, tata letak, dan gaya lainnya.

Contoh Properti CSS

  • color: Mengatur warna teks
  • background-color: Mengatur warna latar belakang
  • font-size: Mengatur ukuran teks
  • margin: Mengatur jarak luar elemen
  • padding: Mengatur jarak dalam elemen

Contoh Kode CSS

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}
h1 {
    color: #003366;
}
